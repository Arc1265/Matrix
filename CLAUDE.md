# Matrix — правила проекта

## Правило версионирования файлов

- Версия ставится ПРЕФИКСОМ в начале имени файла: `FIX3_1_MatrixRegressionChannel_TREND_REGIME.mq5`.
- Небольшие правки (калибровка, багфиксы, мелкие доработки) — суффикс к текущему номеру: `FIX3_1`, `FIX3_2`, ...
- Крупные изменения (новая логика, новая архитектура) — следующий номер по порядку: `FIX4`, `FIX5`, ...
- Каждая версия — отдельный файл; предыдущие версии из репозитория не удаляются (история версий хранится файлами).
- Внутри файла `#define MRC*_VERSION` и шапка-комментарий должны совпадать с именем файла.

## Контекст проекта

Индикаторы MQL5 (MetaTrader 5) на базе полиномиальной (квадратичной) регрессии:
канал Trend ± K·sigma, режимный движок (флэт / тренд вверх / тренд вниз),
входы в начале тренда, выходы для фиксации профита.
Целевые инструменты: крипто-CFD (ETHUSD, BCHUSD) и XAUUSD на FTMO, ТФ M5.

Требования к коду:
- решения только по закрытому бару, сигналы не перерисовываются;
- обязательный CSV-журнал решений (QA) в MQL5\Files;
- HUD-панель состояния на графике.
